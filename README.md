# Premium Store - Responsive Landing Page

## Project Description
A fully responsive landing page for an online store featuring a modern design, product showcase, and mobile-friendly layout.

## Features
- ✅ Responsive design that works on desktop, tablet, and mobile devices
- ✅ Clean and attractive visual design with gradient hero section
- ✅ Navigation menu with smooth scrolling
- ✅ Product section with 4 featured product cards
- ✅ Hover effects on product cards and buttons
- ✅ Professional footer with multiple sections
- ✅ Bootstrap 5.3.0 integration for enhanced responsiveness
- ✅ Well-commented and clean code

## Files Included
1. **index.html** - Main HTML file with semantic structure
2. **styles.css** - Complete CSS styling with responsive media queries
3. **DESIGN_DOCUMENTATION.md** - Comprehensive design documentation
4. **README.md** - This file

## Technologies Used
- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Bootstrap 5.3.0 (CDN)
- Google Fonts (Segoe UI fallback)
- Unsplash API for product images

## Responsive Breakpoints
- **Desktop**: 1200px and above (4-column product grid)
- **Tablet**: 768px - 1199px (2-column grid, hamburger menu)
- **Mobile**: Below 768px (1-column grid, stacked layout)
